Plone in React
==============

### Installation

**Prerequisites**
* [Node.js==6.9.1](https://nodejs.org/)
* [Git](http://git-scm.com/)

**Install dependencies**

    yarn

### Development

    yarn dev

## Testing

    $ yarn test
    $ yarn test:unit
    $ yarn test:e2e

If chromedriver could not be found, make sure to update with:

    ./node_modules/protractor/bin/webdriver-manager update
